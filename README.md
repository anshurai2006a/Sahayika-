# Sahayika - Empowerment Platform for Women

Sahayika is a comprehensive, production-ready digital solution designed to empower women socially and economically. It addresses critical barriers in livelihood access, skill development, legal awareness, and digital safety.

## üöÄ Features

### 1. Safety Command Center
- **Instant SOS:** One-tap emergency trigger that shares live location with trusted contacts and authorities.
- **Walk With Me:** Virtual companion for monitoring commutes.
- **Fake Call Generator:** Discreet tool to exit uncomfortable situations.
- **Digital Safety:** Guides and resources for protecting online presence.

### 2. Career & Livelihood Hub
- **Curated Job Board:** Access to roles from women-forward and inclusive companies.
- **Filtering System:** Search by category, job type, and remote status.
- **Direct Application:** Seamless simulation of job application processes.

### 3. Skill Development (Learning)
- **Course Catalog:** Expert-led courses in Technology, Leadership, Marketing, and Finance.
- **Bootcamps:** Featured high-impact learning pathways like Full Stack Development.
- **Progress Tracking:** Simulated dashboard for monitoring learning milestones.

### 4. Legal Resource Center
- **Rights Education:** Easy-to-understand guides on Workplace Rights, Family Law, and Cyber Law.
- **Expert Access:** Connect with pro-bono legal advisors for consultations.
- **Rights Handbook:** Downloadable resources for offline awareness.

### 5. Community & Mentorship
- **Discussions:** Safe space for sharing experiences and seeking advice.
- **Events:** Access to webinars, meetups, and workshops.
- **Mentorship:** Connect with industry leaders for guidance.

## üì¶ Getting Started

### Prerequisites
- Any modern web browser (Chrome, Firefox, Safari, Edge).
- A local server (Python, Node.js, or Live Server extension).

### Installation
1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd Sahayika-
   ```
3. Start a local server:
   - **Python:** `python -m http.server 8000`
   - **Node.js (serve):** `npx serve .`

4. Open `http://localhost:8000` in your browser.
